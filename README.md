# 🤖 Hugging Face NLP Lab — Text Generation & Embeddings System

## 🧠 Overview

This project is a hands-on NLP engineering lab built with Hugging Face ecosystem tools.

It demonstrates how modern NLP systems are built using:

- Pretrained Transformer models
- Text generation pipelines
- Sentence embeddings
- Real dataset usage
- Modular ML system design

---

## 🚀 Key Features

### 🟢 1. Text Generation Engine
- Uses Hugging Face Transformers
- Supports temperature-controlled generation
- Experiment framework for comparing outputs

### 🟢 2. Sentence Embeddings System
- Converts text into vector representations
- Computes semantic similarity between sentences

### 🟢 3. Mini NLP System
- Sentiment analysis (Hugging Face pipeline)
- Embedding-based similarity
- Unified NLP analysis interface

### 🟢 4. Dataset Integration
- Uses Hugging Face Datasets Hub (IMDb dataset)
- Demonstrates real-world ML data workflow

---

## 🏗 Project Architecture

```
src/
├── text_generator.py        # LLM text generation engine
├── embedding_service.py     # Sentence embeddings + similarity
├── mini_nlp_system.py       # NLP orchestrator (sentiment + similarity)
├── dataset_demo.py          # Hugging Face dataset usage demo
├── embedding_demo.py        # Embedding experiments
├── mini_nlp_demo.py         # Full system demo
```

##  🧪 How to Run
Install dependencies
pip install -r requirements.txt
Run text generation
python main.py
Run embeddings demo
python -m src.embedding_demo
Run NLP system
python -m src.mini_nlp_demo
